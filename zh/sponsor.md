# 赞助 NodeWarden

NodeWarden 是一个尽量轻量、可自托管、兼容官方 Bitwarden 客户端的密码管理服务端。它长期需要维护兼容性、修复安全问题、跟进 Cloudflare 平台变化、完善备份与恢复体验，也需要持续写文档和处理用户反馈。

如果 NodeWarden 帮你省下了服务器成本、部署时间，或者让你更安心地托管自己的密码库，可以考虑给项目一点赞助。金额不重要，持续的支持本身就很有帮助。

## 赞助说明

赞助不是购买服务，也不会改变项目的开源性质、功能优先级或安全边界。你仍然可以免费使用、部署、修改和反馈问题。

如果你是企业或团队用户，希望支持特定方向，可以优先在 GitHub issue 里把需求背景写清楚；赞助可以表达支持，但具体实现仍会按安全性、兼容性和维护成本来判断。

感谢你愿意支持 NodeWarden。哪怕只是使用、测试、提交问题、补充文档或告诉别人这个项目，也是在帮它走得更稳。
